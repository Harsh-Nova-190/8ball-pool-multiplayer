# 🎱 Multiplayer 8-Ball Pool Game

## 📌 Overview

A physics-based multiplayer 8-ball pool game developed in Unity, focused on realistic ball dynamics, accurate collision handling, and smooth gameplay experience. The project simulates real-world pool mechanics while implementing rule-based game logic for a complete two-player experience.


## 🚀 Features

* Realistic ball physics using Unity’s physics engine
* Accurate collision detection and response between balls and table boundaries
* Cue control system for direction and shot power
* Turn-based gameplay system for two players
* Rule-based logic including fouls, ball types, and win conditions
* Smooth camera and UI feedback for player interaction


## 🛠️ Tech Used

* Unity (2D/3D Physics System)
* C#
* Unity Rigidbody & Collision System
* UI System

## 🧠 What I Learned

* Implemented realistic physics simulation using Rigidbody and force-based mechanics
* Designed collision handling systems for accurate ball interaction and response
* Developed turn-based gameplay logic with rule validation and state management
* Learned how to balance physics realism with gameplay feel for better user experience
* Improved debugging skills while handling edge cases like fouls, ball overlap, and invalid shots
* Structured reusable and modular scripts for scalability and maintainability
* Worked on optimizing physics calculations for smoother performance

## 🧪 Challenges Faced
- Handling realistic ball collisions without jitter
- Maintaining consistent physics across different frame rates
- Implementing accurate rule validation for 8-ball gameplay
